Serial Communication Verilog

This project is a digital design coded in Verilog that turns FPGA development board into a simple user interface which allows a user to generate a variety of text content, which will be sent to a computer via serial (RS232) communication. It involves transmitting a character (a package of bits ASCII coding) from FPGA board to PuTTY terminal on the computer using RS232 UART transmission method.

ATmega8 Signal Processing C

This project is an embedded system coded in C to utilise the ATmega8 microcontroller as a signal processor for an incoming analogue signal. The key tasks this system achieves are: Conversion to digital of the analogue signal provided as a "sound" input; Real time processing (recognition of signal activity and signal features); Computation of basic signal analysis; Generation of a text output (on the LCD screen)
